## Instruções :scroll:
Para baixar as dependências:

``
yarn
``

Rodar os testes:

**Crie um banco chamado "gostack_desafio09_tests" para rodar os testes**

``
yarn test
``

Rodar a aplicação:

**Crie um banco chamado gostack_desafio09 para acessar as rotas**

``
yarn dev:server
``
